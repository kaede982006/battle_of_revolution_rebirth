# Battle of Revolution Rebirth

## 테스트 환경

**본 게임은 리눅스 환경에서만 동작합니다.** 윈도우나, 맥 OS의 경우 별도의 리눅스 가상머신을 이용하여 이 게임을 구동할 수 있습니다.
- OS: Ubuntu 24.04.3 LTS x86_64
- CPU: 12th Gen Intel i5-12400F (12) @ 5.600GHz 
- GPU: NVIDIA GeForce RTX 3060 
- Memory: 15831MiB 

## 게임 설명

과거의 게임이 다시 돌아왔습니다. 혁명적 전투라는 게임은 작자가 고2 시절 C언어를 처음 배웠을 때 테스트용으로 만들었던 일종의 시험용 텍스트 게임이였습니다. 게임의 스토리는 소련의 한 스파이 병사가 파시스트와 맞서, 적진을 무너뜨리고 승리를 쟁취하는 스토리입니다.  처음엔 설계적으로, 프로그래밍적으로 미흡했습니다. 프로젝트는 텍스트 파일을 생성하는 등 별도로 리소스를 관리하는 것이 아닌 단일 파일 내에 텍스트를 배치했었습니다. 파일 자체가 효율적으로 분할되어 있지 않았고 단일 파일로 이루어져 있었습니다. 또한, 파이썬을 이용한 이미지 표시기를 이용하였다는 장점도 있었지만, 프로그램이 표시하는 속도가 너무 오래 걸려 게임플레이에 차질을 유발했습니다. 오로지 C언어 파일 하나와 Python 파일 몇 개로 구동되던 게임이였습니다.

지금은 달라졌습니다. 작자는 게임을 동적 라이브러리를 이용해 소스코드를 모듈화하여 시각적이고 효율적으로 재설계하였습니다. 프로젝트는 이제 텍스트도 별도로 분리하였고, 그것을 파일로써 로드하는 기능을 갖췄습니다. 파이썬 이미지 표시기 대신, 터미널에 이미지를 직접 표시하는 기능을 이용하여 속도를 개선하였습니다. 또한 시각적으로 고정된 영역에서 편안한 게임을 즐길 수 있도록 만들었습니다. 또한 컴파일 언어인 C와 C++를 이용하여 언어 융합을 실천하였습니다.

## 게임 빌드 방법

해당 명령어 집합을 터미널에서 구동하세요. 실행 환경은 위의 [테스트 환경](##테스트환경)과 같습니다.
```
sudo apt install ffmpeg git make build-essential
git clone https://github.com/kaede982006/battle_of_revolution_rebirth.git
cd battle_of_revolution_rebirth
./install_resource.sh
make
```
